calculation-of-profit
=====================

Расчет прибыли

Данная программа предназначена для расчета прибыли на этапах жизненного цикла продукции.

Стандартное изменение объема продаж во времени можно изобразить в виде кривой, приведенной в главном окне программы. 
Эту кривую нужно аппроксимировать. Аппроксимация кривой изменения объема продаж товара во времени, приведенная в главном окне, в виде отрезков прямых на каждом из этапов жизненного цикла позволяет обоснованно разбить его на 4 составляющие (опираясь на данные Бостонской консультационной группы).

В данной программе предлагается ввести предполагаемые длительности этапов жизненного цикла и предполагаемые объемы продаж в ключевых точках.
Так же в программе требуется ввести цену изделия и издержки при его создании.
Программа расчитаем прибыль на каждом из этапов жизненного цикла изделия, суммарную прибыль и построит график зависимости объема продаж от времени.
